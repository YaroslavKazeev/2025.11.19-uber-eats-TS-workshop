# 🍔 Uber Eats Clone - TypeScript Workshop

![Uber Eats Demo App](https://github.com/user-attachments/assets/94af8bc3-b77a-4c4f-8f84-cf81ed3079e0)

A modern, responsive food delivery application demo built for the **Hack Your Future** TypeScript workshop. This project demonstrates how to build a complex UI using TypeScript, React, and Uber's Base UI system.

## 🚀 Features

- **Restaurant Browsing**: Clean and intuitive interface to explore various restaurants.
- **Menu Management**: View detailed menus with categories and item descriptions.
- **Shopping Cart**: Real-time cart updates with price calculations.
- **Responsive Design**: Fully optimized for both desktop and mobile views.
- **Type Safe**: Built from the ground up with TypeScript for a robust developer experience.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) (v18)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI Framework**: [Base UI](https://baseui.design/) by Uber
- **Styling Engine**: [Styletron](https://www.styletron.org/)
- **Build Tool**: [Create React App](https://create-react-app.dev/) (react-scripts)

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Recommended version LTS)
- [npm](https://www.npmjs.com/) (Should be included with Node.js)

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/YaroslavKazeev/2025.11.19-uber-eats-TS-workshop.git
   ```

2. **Navigate to the project directory**:

   ```bash
   cd 2025.11.19-uber-eats-TS-workshop
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

## 💻 Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm run build`

Builds the app for production to the `build` folder.  
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm test`

Launches the test runner in the interactive watch mode.
